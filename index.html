FarmaCoft Sistema Web de Gestión y Ventas para Farmacias
Documento de Proyecto

Automatización, Control de Inventario y Ventas POS

FarmaCoft es un sistema web diseñado para optimizar el control de stock, emitir comprobantes, evitar pérdidas por vencimiento de medicamentos y agilizar la atención al cliente en farmacias físicas.

Ver Módulos

Objetivos Clave

Alertas de Stock y Vencimiento

Notificaciones automáticas para productos con bajo stock o próximos a vencer en 30 días, evitando multas y pérdidas.

Venta Rápida

Punto de venta dinámico con lector de código de barras, cálculo de IGV, emisión de tickets PDF y control de recetas médicas.

Reportes Estratégicos

Dashboard con métricas de ventas del día, productos más vendidos y reporte de ganancias reales (Venta - Compra).

Módulos y Funcionalidades

1. Seguridad y Roles
  • Control de acceso (Login / Logout).
  • Administrador: Acceso total y reportes.
  • Vendedor: Ventas y consulta de stock.
2. Inventario y Productos
  • CRUD completo de medicamentos.
  • Gestión de laboratorios y categorías.
  • Marca si "Requiere Receta Médica".
3. Punto de Venta (POS)
  • Carrito de compra rápido.
  • Boletas y Facturas en PDF.
  • Adjunto obligatorio de foto/PDF de receta.
4. Clientes
  • Registro de clientes con DNI / RUC.
  • Opción de "Cliente General" para ventas rápidas.
  • Historial de compras por cliente.
5. Dashboard y Reportes
  • KPIs: Ventas del día, ganancias y vencimientos.
  • Filtros por rango de fechas y productos TOP.
